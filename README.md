1. Установка

Python >= 3.10

- Установка виртуального окружения
pip install virtualenv, активировать

- Установить зависимости:
pip install -r requirements.txt"

2. Запуск автотестов:

pytest test_ui -s -v --headed (для запуска в интерфейсе)
pytest test_service_api.py -s -v

3. Структура тестов:
   -UI-тесты
Авторизация:
успешная и неуспешная (неверный логин/пароль, пустые поля)

Форма услуг:
проверка наличия всех элементов
позитивные граничные значения для price, quantity, name
проверка расчёта НДС и итоговой суммы
негативные сценарии (ошибки валидации, пустые/некорректные поля)

CRUD операции:
создание, редактирование, удаление услуг

-API-тесты

Создание услуги:

позитивные сценарии с корректными данными
проверка расчёта НДС и итоговой суммы
реалистичные данные и граничные значения

Получение услуги:
существующие, несуществующие, некорректные ID

Обновление услуги:
корректное обновление, изменение цены, проверка пересчёта НДС
попытка обновления несуществующей услуги

Удаление услуги:
успешное удаление, двойное удаление, удаление несуществующей услуги
Негативные сценарии:
отсутствие авторизации / неверный токен

некорректные типы данных, пустые поля, превышение лимитов
отрицательные, нулевые и переполняющие значения
